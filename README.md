## Features

- **Free Play** - Interactive virtual keyboard with MIDI support and laptop keyboard input
- **Structured Learning** - ABRSM Grade curriculum from beginner to Grade 8
- **Music Theory Visualizers** - Circle of Fifths, Chord Builder, Scale Visualizer, Interval Visualizer, Rhythm Visualizer, and Chord Progression Player
- **Song Player** - Learn songs with falling notes visualization
- **Composer** - Sheet music editor with AI-powered transcription from PDF

## Tech Stack

- [Next.js](https://nextjs.org/) 16
- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/)
- [Tone.js](https://tonejs.github.io/) for audio synthesis
- [Zustand](https://zustand-demo.pmnd.rs/) for state management
- [Anthropic Claude API](https://www.anthropic.com/) for sheet music transcription

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Usage

### Keyboard Controls

- **White keys**: A, S, D, F, G, H, J, K, L
- **Black keys**: W, E, T, Y, U, O, P
- **MIDI**: Connect a MIDI keyboard to access the full immersive experience
